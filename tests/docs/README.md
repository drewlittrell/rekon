prose-assertion tests guarding snapshot-bannered strategy memos were

the guard, and the pattern was retired by the WO-5 documentation
authority rules ("Do not add new tests/docs prose-assertion tests").
The deletion list is in the WO-7 commit; the enrollment-debt burn-down
list is in the WO-7 completion summary.
