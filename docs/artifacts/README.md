# Artifacts

Artifacts are typed Rekon outputs with schema versioning, producer metadata, input refs, and provenance.

Generated artifacts live under `.rekon/` by default.

Read the [artifact model](./index.md) for the artifact layers and the
producer/consumer guide.

See [artifact-contract.md](./artifact-contract.md) for the repository-wide
contract enforced by Rekon's CLI smoke tests.
