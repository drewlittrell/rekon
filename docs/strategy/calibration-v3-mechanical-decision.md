## Purpose pin (Part 1, class 1)

WO-9 skips type-only imports for layer law because type dependencies do
not violate runtime layering. Reachability has the OPPOSITE purpose:
type-only consumption is consumption, so `typeOnly` edges are references
and a file whose exports are consumed only as types is alive. The two
detectors read the same facts with opposite filters, each cited to its
purpose. (The counters show the scale: ~4,575 type-only references on
mentor alone were already counting as usage; the counter makes it
visible.)

## The four classes as landed (each counted)

- **Type-only (class 1):** already reference-semantics by construction;
  now counted (`typeOnlyReferences`) and contract-tested.
- **Barrel (class 2):** a file with reexport facts amounting to at least
  half its exports is a conduit, exempt from unreferenced-export
  findings; reachability already flows through it. Counted
  (`barrelExemptions`: mentor 84, codebase-intel 42, simulacrum 34,
  figma-ds 4).
- **Generated (class 3):** header markers in the first ten lines
  (`@generated`, AUTO-GENERATED, DO NOT EDIT, "Generated by") arrive as
  a provider `content_signal`; declared codegen globs ride
  `.rekon/scan-scope.json` `generatedGlobs` (WO-10 replace-semantics
  shape, default empty). Counted (`generatedExemptions`).
- **Factory (class 4):** exempt ONLY where the grammar declares the
  signal - the `Factory` fileType role, ratification-dependent. Files
  without a deterministic signal STAND and are reported as residual:
  `createPlanAssemblyTaskRunner.ts` (mentor) and `main-assembly.ts`
  (codebase-intel) are the two named residuals. No prose guessing.

## Part 2: the operator-topology compile gap, fixed

`compileEffectiveGrammar` set `topologies` only for archetype and
advisory packs; overlay and operator-override topology never compiled
in. Fixed at both sites, and the detector gained the eligibility gate it
always should have had: only `findingsEligiblePackIds` packs' topology
backs findings - which also closes a latent leak where advisory-pack
topology, had it ever reached the map, would have fired findings.
Regression-tested both ways.
